# Project Summary

## Enterprise Retail Intelligence Lakehouse

This project is a portfolio-grade retail data platform that modernises traditional grocery inventory and sales reporting into a cloud-style Lakehouse solution.

## Key Features

- Multi-source retail ingestion design
- Bronze, Silver and Gold Medallion Architecture
- PySpark transformation logic
- Fabric Warehouse SQL model
- Power BI dashboard specification
- Demand forecasting
- Stockout prediction
- Inventory reorder recommendations

## Target Roles

- Data Analyst
- BI Analyst
- Analytics Engineer
- Data Engineer
- Data & AI Consultant
